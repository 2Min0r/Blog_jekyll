  2. 프로그래머 첫걸음 15, 16, 17

    • 절대경로와 상대경로

      • 절대경로: /home/bernie와 같이 /로 시작함
      • 상대경로: 현재 작업 디렉터리에서 시작함. /로 시작하지 않을 경우를 말함(❕현재 작업디렉터리: pwd)

    • 플래그

      • --나 -로 표시하며, 플래그를 명령에 추가하면 배시는 플래그의 값을 True로 설정하고, 명령의 작동방식을 변경한다.

    • 환경 변수

      • 운영체제에서 저장한 변수로, 컴퓨터 이름이나 사용자의 이름 같은 정보를 담고 있음.

      • 배시에서 새 환경변수를 만들 수 있으며, 환경변수를 읽을 때는 변수 이름 앞에 $를 붙여야 한다. 이렇게 만들어진 환경변수는 배시창을 나가는 즉시 사라진다.

        export x=100
echo $x			# 100

    • sudo 명령

      • 루트 사용자로 로그인함으로써 컴퓨터의 보안을 위험하게 하지 않고도 루트 사용자의 권한을 임시로 사용할 수 있는 시스템

        $ sudo echo Hello, World!

    • grep 명령

      • grep Beautiful zen.txt : 일치하는 단어를 찾아 그 행 전체를 출력함

      • grep -i beautiful zen.txt: -i 플래그 사용시 대소문자 무시

      • grep -o beautiful zen.txt: -o 플래그 사용시 해당 단어만 출력

      • 파이썬에서는

        import re
line = "Beautiful flower"
matches1 = re.findall("Beautiful", line)
matches2 = re.findall("beautiful", line, re.IGNORECASE)		# 대소문자 무시

    • 캐럿문자(^)와 달러기호($)

      • grep ^If zen.txt: 문자열의 맨 처음에 나타날 때 일치할 경우

      • grep idea.$ zen.txt: 문자열의 맨 끝에 나타날 때 일치할 경우(이때 마지막이 온점(.)으로 끝나지 않을 경우에는 생략해야 함)

      • 파이썬에서는

        import re
zen = "생략"
m = re.findall("^If", zen, re.MULTILINE)
print(m)
m = re.findall("idea.$", zen, re.MULTILINE)
print(m)

    • 대괄호([])

      • echo Two too. | grep -i t[ow]o: [ow] 중 하나만 있어도 검색

      • 파이썬에서는

        import re
string = "Two too."
m = re.findall("t[ow]o", string, re.IGNORECASE)
print(m)

    • 숫자찾기([[:digit:]])

      • echo 123 hi 34 hello? | grep [[:digit:]]: 숫자를 찾음

      • 파이썬에서는

        import re
line = "123 hi 34hello"
m = re.findall("\d", line, re.IGNORECASE)

    • 반복(*)

      • *은 애스터리스크라고 하며, '앞에 있는 것이 0번 이상 나타난다.'라는 뜻이다.
      • echo two twoo not too. | grep -o two*일 경우 two와 twoo가 결과로 출력됨

    • 마침표(.)

      • 어떤 문자와도 일치하므로, 애스터리스크를 함께 사용하면, echo __hello__there | grep -o __.*__의 경우 __hello__가 출력됨

        • 애스터리스크는 적극적(greedy)이므로, 일치하는 단어를 최대한 많이 찾으려고 함. 밑줄 사이에 단어가 여러 개인 다음과 같은 경우 __hello__there__I__am__mina mina를 제외한 전체를 출력함

        • 파이썬에서는 소극적 일치를 사용할 수 있음. 즉, 두번째 밑줄을 만나는 경우 멈춤

          import re
t = "__one__ __two__ __three__"
results = re.findall("__.*?__", t)
for match in results:
    print(match)			# __one__
    						# __two__
        					# __three__

    • 이스케이프(\)

      • echo I love you | grep \you

      • 파이썬에서는 you가 안 된다..?😢

        import re
line = "I love $"
m = re.findall("\\$", line, re.IGNORECASE)
print(m)
