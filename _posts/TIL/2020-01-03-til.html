  2. 프로그래머 첫걸음 17, 18

    • 패키지 매니저: 다른 프로그램을 설치하고 관리하는 프로그램으로, 실제 프로그램을 구성하는 파일 외에도 소프트웨어의 이름, 버전 번호, 의존성 같은 데이터에 대한 데이터(메타데이터)를 포함한다.

    • pip: 파이썬 패키지 매니저

      • [PyPI][http://pypi.python.org/pypi]: pip로 내려받을 수 있는 패키지 목록
      • == 사용시 원하는 버전 내려받음, 미사용시 최신버전

    • Flask: 웹 서버와 웹 사이트를 빨리 만들 수 있는 모듈

    • 스냅샷(snapshot): commit으로 바뀐 내용이 기록된 상태

    • 가상환경

      • python -m venv myvenv		    # 가상환경 만들기
	source myvenv/Scripts/activate	# 가상환경 실행
	. myvenv/Scripts/activate		# 가상환경 실행
	deactivate						# 가상환경 끄기

    • beautifulsoup: 웹 스크래퍼를 만들 때 사용하는 모듈

    • 스택의 메서드: is_empty, push, pop, peek, size

      • peek: 스택의 맨 위에 있는 요소를 반환하지만, 제거하지는 않음

    • 큐의 메서드: enqueue, dequeue, is_empty, size

  1. 프로그래머의 자세 - 4

    • 미숙한 프로그래머와 훌륭한 프로그래머를 나누는 기준은 코드와 자료구조 중 무엇을 더 중시하는가에 달려있다. 미숙한 프로그래머는 코드에 신경을 쓰지만, 훌륭한 프로그래머는 자료구조와 그들의 관계에 신경을 쓴다.
